# My Business Command Center — Master Prompt (Chat version)

Fill in every [BLANK], then paste the whole prompt into a new Claude chat.
Model tip: switch to **Sonnet** for this build — it's a bigger job than the exercises.

---

**Role:** You are an expert web designer and small-business operations assistant. You build clean, branded, single-file HTML dashboards.

**Instructions:** Build me a one-page "Business Command Center" as a single self-contained HTML artifact. It is the home screen for my business, branded for:

- Business name: [YOUR BUSINESS NAME]
- What I do: [ONE LINE — e.g. "hand-poured soy candles, sold online and at markets"]
- Brand colors: [2–3 COLORS, e.g. "blush pink, cream, deep wine"]
- Vibe: [e.g. "warm, feminine, professional — not corporate"]

**Steps:**
1. A header with my business name, today's date, and a one-line welcome in my voice.
2. Three tool cards, side by side. Each card has a title, a one-line description, a **"Copy run prompt"** button, and an empty result panel below it where the result will appear when I run it:
   - **Morning Routine · Dashboard** — run prompt to copy: "Run my morning briefing and deliver it to my command center dashboard."
   - **Bank Statement Analysis · Dashboard** — run prompt to copy: "Analyze the bank statement I've uploaded and deliver the cash-flow dashboard to my command center."
   - **Instagram Dashboard · Dashboard** — run prompt to copy: "Analyze the Instagram export I've uploaded and deliver the audit to my command center."
3. A "Quick Draft" helper section: a short cheat-sheet showing my two writing frameworks (RTF for quick asks, CARE for on-brand content) with a fill-in template for each, styled as copyable text.
4. A footer with my business name and "Powered by my own AI command center."

**End goal:** One beautiful branded page I open every morning: I click a card's copy button, paste the run prompt into this chat, and the matching skill fills that card's result panel — so my briefing, my money, and my marketing all live on one screen.

**Narrowing:** Single HTML file only. No external images, fonts may load from Google Fonts. No login, no fake data — result panels start empty with a gentle "run me" hint. Never invent numbers; panels only show what my skills deliver. Keep the whole page readable on a laptop without horizontal scrolling.

---

## After it builds

1. Install the three **chat skills** (from the workshop downloads) under Settings → Capabilities → Skills.
2. Click a card's "Copy run prompt" button, paste it into the chat, and watch the result land on your dashboard.
3. Iterate out loud: "make the header bigger," "swap wine for terracotta," "add a fourth card for [X]." That's the skill you learned in hour one — keep prompting.
